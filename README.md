## 2. 기능
이하는 현재 모두 구현된 기능들이다.  
Real-Time 적용이 지원되는 항목은 `R`이 붙는다.

### 2 - 1. 계정 및 인증 🔐
* 로그인/회원가입

* 소셜 로그인
  * Google
  * Github
 
### 2 - 2. 유저 👱‍♂️
* Display name 수정 - `R`

* Profile photo 업로드/삭제 - `R`

* 작성한 트윗들 모아보기

### 2 - 3. 트윗 💦
* 트윗 작성 및 사진 업로드 - `R`

* 트윗 수정 및 삭제 - `R`

* 좋아요 기능 - `R`
